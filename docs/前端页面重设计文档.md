# EduSmart 前端页面重设计文档

## 一、现状分析

### 1.1 已实现的后端API（缺少前端页面）

| API路径                         | 功能               | 前端状态          |
| ------------------------------- | ------------------ | ----------------- |
| `/api/obsidian/knowledge-base`  | Obsidian知识库索引 | ❌ 无页面         |
| `/api/obsidian/search?q=`       | 知识库全文搜索     | ❌ 无页面         |
| `/api/obsidian/note?path=`      | 获取单个笔记详情   | ❌ 无页面         |
| `/api/obsidian/graph`           | 知识图谱数据       | ⚠️ 有视图但未对接 |
| `/api/obsidian/status`          | 题库状态统计       | ❌ 无页面         |
| `/api/obsidian/stats`           | 知识库统计         | ❌ 无页面         |
| `/api/obsidian/notes/write`     | 写入笔记到Obsidian | ❌ 无页面         |
| `/api/rag/search?q=`            | RAG向量智能检索    | ❌ 无页面         |
| `/api/rag/chunks/:id`           | 获取RAG分块详情    | ❌ 无页面         |
| `/api/rag/status`               | RAG状态            | ❌ 无页面         |
| `/api/ai-learning/agent-reason` | Agent学习推理      | ❌ 无页面         |
| `/api/diagnostic/*`             | 智能诊断           | ⚠️ 有基础视图     |
| `/api/closed-loop/*`            | 学习闭环           | ⚠️ 部分实现       |

### 1.2 需要新增的前端页面

1. **Obsidian知识库中心** - 浏览、搜索、查看笔记
2. **RAG智能问答** - AI驱动的知识检索与问答
3. **Agent学习中心** - LLM推理路径可视化
4. **学习画像详情** - 完整的个性化分析
5. **知识图谱交互** - D3.js可视化图谱
6. **全功能闭环面板** - 采集→诊断→规划→练习→沉淀→复习

## 二、整体UI设计语言

### 2.1 色彩体系

```
主色: #635bff (紫蓝)
辅色: #00c9a7 (青绿)
强调: #ff6b6b (珊瑚红)
背景: #f7f8fc (浅灰蓝)
卡片: #ffffff
文字: #10203f (深蓝黑)
副文: #66708a (灰蓝)
边框: #e8ecf1
```

### 2.2 字体与间距

- 标题: `font-weight:700`, `letter-spacing:-0.02em`
- 正文: `font-size:14px`, `line-height:1.7`
- 卡片圆角: `12px`
- 按钮圆角: `8px`
- 阴影: `0 1px 3px rgba(16,32,63,0.08)`, hover: `0 4px 16px rgba(99,91,255,0.12)`

### 2.3 组件风格

- 卡片: 白色背景，微阴影，悬停上浮
- 按钮: 渐变主色，圆角8px
- 输入框: 浅灰边框，聚焦紫蓝发光
- 标签: 圆角胶囊形
- 图标: SVG内联，20px基准
